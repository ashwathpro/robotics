Hi ,

First thing, there are many ways of obstacle detecition.

1. LED+LDR: You can get a maximum range of 3cm because of the effect of external light will affect it mostly because of the visible light of LED. So its range is less. You can use a LDR+reistor and use comparator to identify if there is obstacle or not.

2. IR LED+ PhotoDiode(Photo Transistor): In this case range is more extended. I got it working upto 7cm. But this also have the effect of external light because external light also contains IR and power of IR led matters also

3. Laser Diode+Photo Transistor: This has a good range,but it requires exact line of sight of propagation, especially used for data transmission, but adjusting line of sight for photo transistor is a great problem. Laser is a coherent and sharp which makes it a good option, but u had to hack a laser diode and make it working accurately in line of sight. This is a difficult task, otherwise it would have been a better option

4. 38KHz IR+TSOP1738: This technique is used in TV remote for transmission of signals. For this you have to generate a 38KHz signal mostly using 555 and connect the complement of the output of the serial port to the RESET of 555, but limit the maximum baud rate should be 1.2KBPS, see the datasheet of tSOP1738. TSOP have inbuilt filters to detect only 38KHz signals. It is used in obstacle detection by using 555 to generate 38Khz and receiving it using TSOP. Its range varies around 2m, but depends on the color of the surface. So you can detect distances in the range of meters. The IR 38KHz will come back if there is some obstacle and when 38KHz fall on this the output of TSOP become LOW. So if you don't want to mess up with 555 timings and adjustments, go for counter. Connect counter output to the IR LED and run the counter to produce 38KHz. Suppose if you want to transmit '0' apply 38KHz for a time of the baud rate of the computer (1/baud time). Apply sequence in proper format '0" first then the 8 bits then one stop bit.

Go through net you will get more ideas, some information will be available in my first book
